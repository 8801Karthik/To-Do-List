# 📝 Todo List App (React + Vite)

A modern Todo List application built using **React** and **Vite**, featuring task management, drag-and-drop reordering, and persistent storage using localStorage.

---

## 🌐 Live Demo

👉 https://your-vercel-link.vercel.app

---

## 🚀 Features

- ➕ Add new tasks
- ❌ Delete tasks
- ✅ Mark tasks as completed
- 📌 Separate sections for active and completed tasks
- 🔄 Drag and drop to reorder tasks within each list
- 💾 Persistent storage using **localStorage** (tasks remain after refresh/close)
- 📱 Responsive UI for mobile and desktop

---

## 🛠️ Tech Stack

- React (Hooks: useState)
- Vite
- JavaScript (ES6+)
- CSS
